Häufige Fragen

Vor dem Bestellen.

Ist das Set für mich, wenn ich noch nie gespielt habe?
Ja, das ist genau der Punkt. Das Set ist für Spieler, die anfangen oder gerade angefangen haben. Wenn Sie schon Vereinserfahrung haben und in einer Liga spielen, brauchen Sie wahrscheinlich etwas Spezifischeres. Schreiben Sie uns kurz, in welchem Bereich Sie unterwegs sind, und wir sagen Ihnen ehrlich, ob das Set passt oder nicht.
Kommt das Set fertig zusammengebaut?
Ja. Die Beläge sind auf das Holz geklebt, der Griff hat schon Kantenband, die Schutzhülle ist dabei. Sie packen aus und können spielen. Das ist der Unterschied zu allem, was Sie als „Komplettschläger" im Supermarkt finden, wo Sie technisch zwar einen fertigen Schläger haben, aber keinen, der wirklich zum Spielen taugt.
Wie lange hält so ein Set?
Bei regelmäßigem Vereinstraining (2-3 mal pro Woche) etwa 12 bis 18 Monate für die Beläge. Das Holz hält deutlich länger, oft mehrere Jahre. Wenn die Beläge nachlassen, können sie einzeln ersetzt werden, das Holz bleibt.
Kann ich später bessere Beläge kaufen und einfach umkleben?
Genau so ist das Set gedacht. Wenn Sie nach einem Jahr merken, dass Sie einen schnelleren Belag wollen, lösen Sie die alten ab und kleben neue drauf. Das Maxwill V3 Carbon-Holz darunter passt zu einer breiten Spanne an Belägen, von Allround bis offensiv.
